[*****INSTRUCTION*****]

•How to run the backend:
  ASP.NET CORE WEBAPI based on .NET 8
  1.launch the run_server.bat
  2.go to http://localhost:7092/swagger/index.html to check the swagger ui page


•How to run the frontend:
  The frontend project based on React 19.1.0
  1.launch the run_client.bat
  2.the browser will automatically open a new page to redirect http://localhost:3000/
  3.if it doesn't work, please open the folder datacom-jobapplicationtracker-client with VSCode and run "npm install" and "npm start" to launch it

•Any assumptions made:
  The data is stored in memory, so it will be lost after relaunching the backend service
